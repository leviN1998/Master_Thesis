# Blender Scenes

Here is the place to save all relevant blender files.
The files currently included are:
* pingpongtable.fbx: downloaded model of a pingpongtable
* lab.blend: reconstruction of the table-tennis robot lab with accurate measurments.
* throw_ball_physics.blend: A physically simulation of a ball that is thrown. Not used for the simulation because simulating gravity is overkill
* basic_cube.blend: Just a cube that can rotate. Was used for the first iterations of the simulator
* ball.blend: The model of the Andro poly 2s ball. this can be used to inlcude the ball into any scene
* ball_moving.blend: ball placed in a scene with camera that can be used for the event simulation. The camera is placed at the same location as in the real
lab. So it can be used to produce accurate simulations.
* ball_moving_aligned.blend: Newest iteration of the scene, where the camera is placed at the axis, so we dont have any artifacts and distortions. More simple than the other version, maybe transition back.
