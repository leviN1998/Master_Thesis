# Output folder

Here are all files generated by the simulator during simulations. <br>
For now, there are some files that where used for progress reports etc. but in theory there shouldn't be any permanent files here <br>
The simulations that are saved at the moment are:
* ball_event_test: first test run of a spinning ball with event generation
* different_framerates: event-videos collected using different framerate settings
* temp: a folder where the temp image of blender is rendered into
